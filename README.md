# Feline The Modern Scratch Editor

## Why you should use Feline?

 - Feline use the [LitJS](https://lit.dev), It's a framework faster than React (the framework for the original version of Scratch and other distributions)

 - Feline have powerful package and extension manager

## Setup And Onboarding

Install the dependencies:

```bash
git clone https://github.com/xiaotiaojiang/feline-lit.git
# 对于中国开发者，请使用 git clone https://wget.la/https://github.com/xiaotiaojiang/feline-lit.git
cd feline-lit
pnpm install
```

## Launch Feline

Start the Feline-Dev server, and Feline will be runing at [http://localhost:3000](http://localhost:3000).

```bash
pnpm run dev
```

Build Feline for yourself or distribute Feline:

```bash
pnpm run build
```

Preview the production build locally:

```bash
pnpm run preview
```
